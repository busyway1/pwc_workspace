# Tax Case Review Skill (조세 판례 분석)

## When to Use

- 조세 판례문(PDF/DOCX) 분석 요청 시
- 세법 해석 관련 질의 시
- 유사 판례 비교 분석 시

## Domain Knowledge

### 한국 조세체계 개요

- **국세**: 소득세, 법인세, 부가가치세, 상속세, 증여세, 종합부동산세
- **지방세**: 취득세, 재산세, 지방소득세

### 판례 분석 구조

1. **사건 개요**: 당사자, 처분내용, 쟁점
2. **관련 법령**: 적용 세법 조항 식별
3. **쟁점 분석**:
   - 납세자 주장 vs 과세관청 주장
   - 법원/심판원 판단 근거
4. **결론**: 인용/기각/일부인용
5. **시사점**: 실무 적용 포인트

### 주요 참조 기관

- **대법원** (Supreme Court): 최종 판결
- **조세심판원** (Tax Tribunal): 행정심판
- **국세청** (NTS): 예규, 질의회신
- **기획재정부**: 세법해석

## Analysis Workflow

1. `doc-processor` MCP의 `parse_pdf` 또는 `parse_docx` 도구로 판례문 파싱
2. 문서 구조 분석 (판결문/결정문 형식 파악)
3. 핵심 쟁점 추출
4. 관련 법령 조항 식별
5. 판결 논거 구조화
6. 실무 시사점 도출

## Output Format

- **판례 요약**: 사건번호, 선고일, 결과
- **쟁점 분석표**: 쟁점별 양측 주장 대비
- **적용 법령**: 관련 세법 조항 목록
- **실무 시사점**: 유사 사안 적용 가이드
